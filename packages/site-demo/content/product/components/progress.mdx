```javascript import
import { Progress, ProgressVariant } from '@lumx/react';
```

# Progress

**Progress notify user that an action is being completed.**

Progress come in two styles: _circular_ and _linear_.

## Circular

```javascript jsx withThemeSwitcher disableGrid
(theme) => (
    <>
        <Progress theme={theme} />
    </>
);
```

## Linear

```javascript jsx withThemeSwitcher disableGrid
(theme) => (
    <>
        <Progress theme={theme} variant={ProgressVariant.linear} />
    </>
);
```
