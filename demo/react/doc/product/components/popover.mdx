# Popover

**Popovers display pieces of UI on top of others. They are positioned from anchor elements.**

## Positions

### Directed

Popovers can be placed to 4 positions :
_TOP_, _RIGHT_, _BOTTOM_, _LEFT_

```javascript jsx withThemeSwitcher
(theme) => {
    // Top
    const anchorTopRef = React.useRef(null);
    const popoverTopRef = React.useRef(null);
    const { 
        computedPosition: topPosition,
        isVisible: isTopVisible
    } = Popover.useComputePosition(Placement.TOP, anchorTopRef, popoverTopRef, true);

    // Right
    const anchorRightRef = React.useRef(null);
    const popoverRightRef = React.useRef(null);
    const { 
        computedPosition: rightPosition,
        isVisible: isRightVisible
    } = Popover.useComputePosition(Placement.RIGHT, anchorRightRef, popoverRightRef, true);

    // Bottom
    const anchorBottomRef = React.useRef(null);
    const popoverBottomRef = React.useRef(null);
    const { 
        computedPosition: bottomPosition,
        isVisible: isBottomVisible
    } = Popover.useComputePosition(Placement.BOTTOM, anchorBottomRef, popoverBottomRef, true);

    // Left
    const anchorLeftRef = React.useRef(null);
    const popoverLeftRef = React.useRef(null);
    const { 
        computedPosition: leftPosition, 
        isVisible: isLeftVisible
    } = Popover.useComputePosition(Placement.LEFT, anchorLeftRef, popoverLeftRef, true);

    const popoverContent =  <div style={{ margin: '20px' }}>Popover</div>;
    const verticalPadding = '60px';
    const horizontalPadding = '80px';

    return (
        <>
            <div style={{ padding: `${verticalPadding} ${horizontalPadding} ${verticalPadding} 0` }}>
                <Chip chipRef={anchorTopRef} theme={theme}>
                    TOP
                </Chip>
                <Popover popoverRect={topPosition} popoverRef={popoverTopRef} isVisible={isTopVisible}>
                    {popoverContent}
                </Popover>
            </div>
            <div style={{ padding: `${verticalPadding} ${horizontalPadding} ${verticalPadding} 0` }}>
                <Chip chipRef={anchorRightRef} theme={theme}>
                    RIGHT
                </Chip>
                <Popover popoverRect={rightPosition} popoverRef={popoverRightRef} isVisible={isRightVisible}>
                    {popoverContent}
                </Popover>
            </div>
            <div style={{ padding: `${verticalPadding} ${horizontalPadding}` }}>
                <Chip chipRef={anchorBottomRef} theme={theme}>
                    BOTTOM
                </Chip>
                <Popover popoverRect={bottomPosition} popoverRef={popoverBottomRef} isVisible={isBottomVisible}>
                    {popoverContent}
                </Popover>
            </div>
            <div style={{ padding: `${verticalPadding} 0 ${verticalPadding} ${horizontalPadding}` }}>
                <Chip chipRef={anchorLeftRef} theme={theme}>
                    LEFT
                </Chip>
                <Popover popoverRect={leftPosition} popoverRef={popoverLeftRef} isVisible={isLeftVisible}>
                    {popoverContent}
                </Popover>
            </div>
        </>
    );
};
```

### Auto

Popovers can also be placed automatically where space is available. They will be placed at the bottom of anchors by default.

```javascript jsx withThemeSwitcher
(theme) => {
    const anchorRef = React.useRef(null);
    const popoverRef = React.useRef(null);

    const { computedPosition, isVisible } = Popover.useComputePosition(Placement.AUTO, anchorRef, popoverRef, true);

    return (
        <div style={{ padding: '60px 0' }}>
            <Chip chipRef={anchorRef} theme={theme} size={Size.s}>
                Anchor
            </Chip>
            <Popover popoverRect={computedPosition} popoverRef={popoverRef} isVisible={isVisible}>
                <div style={{ margin: '20px' }}>
                    Popover
                </div>
            </Popover>
        </div>
    );
};
```

## Offset

Popovers positions can have an offset.

```javascript jsx withThemeSwitcher
(theme) => {
    const anchorRef = React.useRef(null);
    const popoverRef = React.useRef(null);

    const { computedPosition, isVisible } = Popover.useComputePosition(
        Placement.RIGHT, 
        anchorRef, 
        popoverRef, 
        true, 
        {
            horizontal: 30,
            vertical: 60,
        }
    );

    return (
        <div style={{ paddingBottom: '60px' }}>
            <Chip chipRef={anchorRef} theme={theme} size={Size.s}>
                Anchor
            </Chip>
            <Popover popoverRect={computedPosition} popoverRef={popoverRef} isVisible={isVisible}>
                <div style={{ margin: '20px' }}>
                    Popover
                </div>
            </Popover>
        </div>
    );
};
```

## Alignements

In addition to positions, popovers can be aligned to a chosen anchor's side :
_TOP_START_, _TOP_END_, _RIGHT_START_, _RIGHT_END_, _BOTTOM_START_, _BOTTOM_END_, _LEFT_START_, _LEFT_END_, _AUTO_START_, _AUTO_END_,

```javascript jsx withThemeSwitcher
(theme) => {
    // TOP_START
    const anchorTopRef = React.useRef(null);
    const popoverTopRef = React.useRef(null);
    const { 
        computedPosition: topPosition,
        isVisible: isTopVisible
    } = Popover.useComputePosition(Placement.TOP_START, anchorTopRef, popoverTopRef, true);

    // TOP_END
    const anchorRightRef = React.useRef(null);
    const popoverRightRef = React.useRef(null);
    const { 
        computedPosition: rightPosition,
        isVisible: isRightVisible
    } = Popover.useComputePosition(Placement.TOP_END, anchorRightRef, popoverRightRef, true);

    const horizontalPadding = '240px';

    return (
        <>
            <div style={{ paddingRight: horizontalPadding, paddingTop: '60px' }}>
                <Chip chipRef={anchorTopRef} theme={theme}>
                    TOP_START
                </Chip>
                <Popover popoverRect={topPosition} popoverRef={popoverTopRef} isVisible={isTopVisible}>
                    <div style={{ margin: '20px' }}>
                        Popover aligned on start of the top side
                    </div>
                </Popover>
            </div>
            <div style={{ paddingLeft: horizontalPadding, paddingTop: '60px' }}>
                <Chip chipRef={anchorRightRef} theme={theme}>
                    TOP_END
                </Chip>
                <Popover popoverRect={rightPosition} popoverRef={popoverRightRef} isVisible={isRightVisible}>
                     <div style={{ margin: '20px' }}>
                        Popover aligned on end of the top side
                     </div>
                </Popover>
            </div>
        </>
    );
};
```
