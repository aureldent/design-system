# Slideshow

**Slideshows display multiple pieces of featured content on a limited allocated amount of space.**

## Default

```javascript jsx
(theme) => {
    const slideshowWrapperStyle = {
        display: 'flex',
        justifyContent: 'center',
    };

    const slideshowStyle = {
        width: '50%',
    };

    const imageBlockDemoProps = {
        description: 'What an image',
        tags: ['#tag1', '#tag2', '#tag3'],
        title: 'Nice Image',
    };

    return (
        <Slideshow activeIndex={0} hasControls={true} theme={theme} autoPlay={false} groupBy={1} style={slideshowStyle}>
            <SlideshowItem>
                <ImageBlock
                    aspectRatio={ThumbnailAspectRatio.horizontal}
```

## Group by

Slideshows can display groups of pieces of content.

```javascript jsx

    const slideshowWrapperStyle = {
        display: 'flex',
        justifyContent: 'center',
    };

    const slideshowStyle = {
        width: '100%',
    };

    const imageBlockDemoProps = {
        description: 'What an image',
        title: 'Nice Image',
    };

    return (
        <Slideshow activeIndex={0} hasControls={true} theme={theme} autoPlay={false} groupBy={2} style={slideshowStyle}>
            <SlideshowItem>

                    aspectRatio={ThumbnailAspectRatio.horizontal}
```
